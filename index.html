Laravel-swap

Currency exchange rates for Laravel

View on GitHub Download .zip Download .tar.gz

Swap allows you to retrieve currency exchange rates from various services such as Fixer or Yahoo and optionally cache the results.

QuickStart

1) Install via Composer:

$ composer require florianv/laravel-swap php-http/message php-http/guzzle6-adapter

2) Configure the Service Provider and alias:

// config/app.php
'providers' => [
    Swap\Laravel\SwapServiceProvider::class
],

'aliases' => [
    'Swap' => Swap\Laravel\Facades\Swap::class
]

3) Publish the Package configuration

$ php artisan vendor:publish --provider="Swap\Laravel\SwapServiceProvider"

4) Start using it!

// Get the latest EUR/USD rate
$rate = Swap::latest('EUR/USD');

// 1.129
$rate->getValue();

// 2016-08-26
$rate->getDate()->format('Y-m-d');

// Get the EUR/USD rate yesterday
$rate = Swap::historical('EUR/USD', Carbon::yesterday());
